# Offers

## Create an offer

[Watch the walkthrough](https://www.loom.com/share/eb4a1f46455c491383ecdda99d0dad24?sid=87719dd3-1c7b-4ccb-81a7-ea86c17e6c67)

<Steps>
  ### Navigate to the [Brand Dashboard](https://prd-xms-api.combinedcuriosity.io/admin/brands/).
  ### Select the brand you want to create an offer for.
  ### Click on the "Offers" tab.
  ### Click the plus icon next to the section title.
  ### Complete all necessary information in the form.
  ### Click the "Create Offer" button.
  ### Look for the success message confirming the creation of the offer.
  ### The section will refresh, and you will find the newly added offer.
</Steps>

## Edit an exsiting offer

_Note: If an offer has been purchased, you can only edit the name, description, end date, and refundability._

<Steps>
  ### Navigate to the [Brand Dashboard](https://prd-xms-api.combinedcuriosity.io/admin/brands/).
  ### Select the brand you want to edit an offer for.
  ### Click on the "Offers" tab.
  ### Click the pencil and paper icon next to the entry you wish to edit.
  ### Make any necessary changes in the form.
  ### Click "Update Offer" when finished.
  ### A confirmation message will inform you of the successful offer update.
  ### The section will refresh, and you will see the applied changes to the offer.
</Steps>

## Copy an existing offer

<Steps>
  ### Navigate to the [Brand Dashboard](https://prd-xms-api.combinedcuriosity.io/admin/brands/).
  ### Select the brand you want to copy an offer for.
  ### Click on the "Offers" tab.
  ### Click the dual paper icon next to the entry you want to copy.
  ### Make any changes in the form.
  ### Select "Create Offer" when finished.
  ### Look for the success message confirming the creation of the new offer.
  ### The section will refresh, and you will find the new offer.
</Steps>

## Archiving or deleting an offer

_Note: You can only delete an offer if it has not been purchased._

<Steps>
  ### Navigate to the [Brand Dashboard](https://prd-xms-api.combinedcuriosity.io/admin/brands/).
  ### Select the brand you want to archive or delete an offer for.
  ### Click on the "Offers" tab.
  ### Select the file box or trash can icon next to the entry you want to remove.
  ### A confirmation box will appear. Select OK to proceed.
  ### Look for the confirmation message indicating the successful removal of the offer.
  ### The section will refresh, and the offer will no longer be displayed.
</Steps>

## Viewing details about an offer

<Steps>
  ### Navigate to the [Brand Dashboard](https://prd-xms-api.combinedcuriosity.io/admin/brands/).
  ### Select the brand you want to view an offer for.
  ### Click on the "Offers" tab.
  ### Click the magnifying glass icon next to the offer.
  ### The associated package and a list of products grouped by type will appear.
  ### Click the back arrow to return to the offer view.
</Steps>

## What do all these values mean? [#value-in-meaning]

**Associated Package:** The package that the offer is selling.

**Name:** The name of the offer, displayed in the modal for easy identification.

**Description:** A brief description of the offer, visible in the purchase modal when the details link is clicked.

**Price:** The total cost of the offer.

**Number of Installments:** Indicates how many payments the total price will be divided into.

**Months Before Charge:** The duration before the first payment is expected.

**End Date:** The date when the offer ends.

**Refundable:** Determines whether a purchase of this offer can be refunded.

**Is Subscription:** Indicates if the offer involves a recurring charge.

**Shipping Required:** Specifies whether the package associated with the offer contains any physical products, necessitating shipping.
