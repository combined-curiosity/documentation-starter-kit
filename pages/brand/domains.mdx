### Terraform create events domain (`e.<brand domain>.com`).
### Terraform create CollectorJS Domain (`cjs.<brand domain>.com`).
### Manually create XMS-API wrapper subdomains (`buy.`, `cc-api.`).
### Manually create LX wrapper subdomain.
  - Add to existing internal Cloudfront distribution with `<brand>-prd-lx.combinedcuriosity.io`.
  - Add `cc.io` DNS entry in Route53.
  - Add brand domain passthrough distribution of `learn.<brand domain>.com`.
